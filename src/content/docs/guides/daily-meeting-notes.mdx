## Where to find it

1. Open **Daily Notes** from the left sidebar.
2. Click **New Daily Note**.

A new note is created for today and opened directly.

## What you can do

- Set or adjust the **date**.
- Expand **Recent Pull Requests** to load team PR activity around that date.
- Write your daily summary in the **Notes** markdown editor.

## Saving behavior

- Note content is autosaved while you type.
- EM Kit tries to save before you leave the page when changes are pending.

## Manage your notes list

- Use the list page to reopen previous notes.
- Delete outdated notes from the table actions.
