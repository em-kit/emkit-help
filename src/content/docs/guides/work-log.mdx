## Where to find it

Open **Work Log** from the left sidebar.

## Tabs

- **My Work Log**: personal manager achievements.
- **Team Work Log**: team achievements with contributor attribution.

## Add or edit entries

Each entry supports:

- **Date**
- **Achievement / Win**
- **Details** (optional)
- **Links** (GitHub, Jira, and others)

For **Team Work Log**, at least one **Contributor** is required.

## Generate weekly Slack draft

1. Open **Team Work Log**.
2. Click the summary action to generate a draft.
3. Pick week start and end dates.
4. Click **Generate**, then **Copy** to clipboard.

The draft includes manager achievements and team wins.
