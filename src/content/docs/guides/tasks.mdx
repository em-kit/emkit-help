## Where to find it

Open **Tasks** from the left sidebar.

## My Tasks

Use this section for your own manager to-dos:

- Add a task with **Add**
- Edit title inline
- Mark complete with checkbox
- Set priority (`High`, `Medium`, `Low`)
- Delete when done

## Assigned in 1:1s

This section lists action items assigned to you during one-on-ones.

- Edit inline
- Toggle completion
- Set priority
- Open the source 1:1 from the contextual link
