## Where to find it

Open **Analytics** from the left sidebar.

## Available report

Current report type:

- **GitHub Activity Comparison**

This report compares PRs, merged PRs, and reviews across team members.

## Create a reporting

1. Open **GitHub Activity Comparison**.
2. Click **New Reporting**.
3. Select a date range.
4. Optionally enable force refresh to regenerate an existing period.
5. Click **Create**.

## Manage report history

From the report list, you can:

- Open a saved reporting
- Delete a saved reporting snapshot
- Monitor status (`Pending`, `Processing`, `Completed`, `Error`)

## Report detail page

The detail page shows:

- Real-time progress while processing
- Per-member metrics table (PRs, merged PRs, reviews)
- Top performer tag
- Excluded members and reasons
